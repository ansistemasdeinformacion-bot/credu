CREDU - Sistema de Credenciales Institucionales Uniagustiniana

🔧 REQUISITOS:
- Servidor web local (Live Server de VS Code o similar)
- Navegador actualizado (Chrome, Edge, Firefox)
- Archivo docentes.xlsx debe estar en la misma carpeta

🚀 CÓMO USAR:
1. Abre index.html con Live Server (necesario para leer el Excel)
2. Ingresa con tu correo institucional de docente
3. En el asistente, digita tu número de cédula
4. Recibirás confirmación de envío de credenciales de SIGA, KAWAK, SIPA HCM
5. Elige "Sí" si necesitas más ayuda o "No" para salir

📝 NOTAS:
- Las credenciales se envían de forma simulada al correo registrado
- Si la contraseña dice "Contraseña personalizada", se mostrará advertencia
- Si olvidaste tu contraseña personalizada, contacta a an.sistemasdeinformacion@uniagustiniana.edu.co

👩‍💻 TECNOLOGÍAS:
- HTML5, CSS3, JavaScript
- SheetJS (XLSX) para leer Excel
- Diseño responsive
